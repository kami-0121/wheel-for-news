# 🎡 WheelTimer                                                                                      

支援無邊框、透明背景、快捷鍵切換，以及可自訂轉盤項目

---

## ✨ 功能特色
- ✅ **幸運轉盤**：支援自訂項目（每行一個） 
  - 自訂義轉盤顏色
  - 控制計時器時間
- ✅ **快捷鍵**：  
  - `F10` → 切換邊框模式  切換透明度
  - `F9` → 自訂義時間背景透明
  - `F8` → 自訂義時間字體粗體
- ✅ **計時器**：計時器控制器
  - 計時器可以自行加減 
  - `1h` `5m` `1m`
  - `-1h` `-5m` `-1m`
